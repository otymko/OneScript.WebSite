## Конструкторы

  
### Конструктор по умолчанию
    
    
Формирует ГСЧ с возможностью указания начального числа.


  
  
#### Параметры

* *seed*: Начальное число. Последовательность случайных чисел для одного и того же начального числа будет одинакова
